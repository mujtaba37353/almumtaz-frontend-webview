# قائمة تحقق قبول الأدوار (QA)

## AppOwner / AppAdmin
- [ ] تسجيل الدخول
- [ ] لوحة رئيسية بإحصاءات حقيقية
- [ ] إدارة الاشتراكات
- [ ] عرض الحسابات مع عدد المستخدمين الحقيقي

## AccountOwner
- [ ] إعداد بيانات المنشأة (VAT/CR/عنوان)
- [ ] حفظ اعتماد ZATCA
- [ ] إنشاء متجر / مستخدم / منتج ضمن الحدود
- [ ] فتح جلسة وبيع مع ضريبة
- [ ] عرض فاتورة + QR
- [ ] تقارير وفواتير

## StoreAdmin / StoreAccountant
- [ ] منتجات ومبيعات وتقارير للمتجر فقط

## Cashier
- [ ] فتح/إدارة جلسة
- [ ] إضافة للسلة ومنع البيع بدون مخزون كافٍ (رسالة خطأ)
- [ ] طرق دفع: نقدي / بطاقة / آجل
- [ ] إغلاق الجلسة
