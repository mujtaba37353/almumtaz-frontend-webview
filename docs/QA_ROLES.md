# قائمة تحقق قبول الأدوار — Mumtaz ERP

## AccountOwner
- [ ] عملاء/موردون
- [ ] مستودعات + تحويل + جرد
- [ ] أمر شراء → استلام → فاتورة مشتريات
- [ ] بيع POS مع عميل وآجل
- [ ] سند قبض/صرف + أعمار ديون
- [ ] ميزان مراجعة / قائمة دخل / ضريبة
- [ ] CSR + ZATCA credentials

## GeneralAccountant
- [ ] قيود يدوية
- [ ] تقارير مالية
- [ ] ذمم

## StoreAdmin / Cashier
- [ ] جلسة بيع وخصم مخزون
- [ ] منع البيع بدون رصيد
- [ ] اختيار عميل للآجل

## سيناريو ذهبي
1. إنشاء مورد ومنتج ومستودع
2. أمر شراء واستلام
3. بيع من POS
4. التحقق من القيد والمخزون والفاتورة
